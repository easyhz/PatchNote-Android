# 패치노트

### 쉽고 빠른 하자 관리

패치노트는 실시간 하자 정보 관리를 돕는 서비스입니다.

접수자가 하자 정보와 사진을 등록하면, 작업자는 증빙자료를 첨부해 완료처리합니다.

이를 통해 효율적이고 투명한 하자 관리 프로세스를 구현합니다.
